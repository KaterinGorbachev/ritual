## creating new pages 

i am a product UI UX designer and full stack developer with next.js, react, tailwind senior level experience. The project is for a massage salon in Valencia. It is a web site that combines landing page with whatsapp button to make an appointment, FAQs and blog page, page with services and prices. It is a 3 language web site with English, Russian and Spanish versions. Make a prototype of a page for services and prices with html and tailwind that reflects the aesthetic of the salon and colors. Prepare marketing texts for services in 3 languages and sections description for facial, body, manicure and pedicure. There must be a state when data about services and prices did not loaded from data Base, user friendly. As a background between section add custom animation of soap bubbles moving. Use slight gradient working around salon pink palette. The design must be responsive. Check minimum accessibility requirements. The output must be a html with comment in english before each section or UI that can be made as react component. Use given files as example, use only inline tailwind styles and gradients, do not create custom tailwind variables if it can be avoided.

use violet colors as minimum, work with salon primary color --color-blush: #ffadae; and its shadows and gradients, use the soap bubbles as here, and make background similar to nano banana animation between sections but with salon soap bubbles with magnifying glass effect, use shadow for the cards as <li data-testid="service-card" className={
flex flex-col p-2 relative h-full min-h-72 w-full max-w-[350px] overflow-hidden rounded-card border border-blush/10 bg-cream/80 text-start transition hover:shadow-[0_0_0_1px_rgba(218,24,132,.12),0_18px_50px_-24px_rgba(218,24,132,.45)]  ${className}}> add custom elegant thin line, blush effect icon for service in the box with gradient

NEVER use generic AI-generated aesthetics in <frontend_aesthetics> animations for effects and micro-interactions and grid. Do NOT use soap bubbles on this page behind title and description as it is distraction for a user. Use soap animation as a absolute positioned background with transparent background placed between sections. Each service section put into the container with its variant of salon pink color. Then inside the container put cards with services. Use as much layers as needed, so the soap animation is visible but primary role is for texts and service cards.  All pink variant colors must be complemented colors without high contrast. use magenta color only for texts and icons as it is a contrast color to salon pink color.  Make a gradient mix and blend for backgrounds with cyberpunk, techno style but using salon pink color. If there is an error or is in the process of loading, stop animation. Animation must start only when data was loaded not to spend operation and computation to animation, but spend all for a data loading.
